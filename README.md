# BarberHouse [IN DEVELOPMENT PHASE]
## About project
The project is a web application for a barber's salon that allows users to quickly register for a visit and barbers to manage their visits.

## Technologies
- b a c k e n d:
  - **ASP.NET Core Web API**
    - REST API
  - **C#**
- f r o n t e n d:
  - **React**
    - JavaScript
